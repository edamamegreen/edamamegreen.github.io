This is the personal website of Nicki Vance.

I'm a user experience and product designer currently living in Melbourne, Australia.

I run a design consulting firm, Boxing Industries, where I help companies plan and design awesome, usable apps and web tools. I've worked with the United Nations, Adidas, Comcast, Cigna, Intel, and HTC, to name a few.

Moneyhawk is a personal finance app for Luddites that I designed and launched with Kevin Lynagh.

You'll usually find me traveling and working on various side projects in games, animation, and web development.

Games

  • Smash Banana is a driving game where you drive a boxy Volvo (modeled on my own) into bananas.
  • Dogtron is about a herding dog.
  • Flipside is a prototype game mechanic for revealing the path ahead of your character/banana.

Web Development

  • Moneyhawk's website and blog were designed and developed by me.
  • Zappy is a prototype interface for a microwave.
  • The So Fresh 9000 is a prototype interface for a washing machine.
