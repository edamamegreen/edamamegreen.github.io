About

Plant Boss is a game about growing plants and attracting cool critters.

This web game was built with Clojurescript, Rum (a React library), and uses a finite state machine for the control logic.

What's Next?

At the end of January, I'll circulate the game and gauge reactions. I need to come up with some concrete tests to decide whether to keep working on this game.

Recent Updates

Jan 26, 2018

All the art is still stand-ins but I've been bringing in test art and a higher fidelity background so I can get my positioning system worked out.

There are moving critters in the game now! I've added ants and a gecko. The gecko's tongue flicks out but it's so tiny you can barely see it.

I'm creating these spritesheets by hand in Sketch. This isn't easy because you can't see how the frames you're creating add up to the animation until you've exported the image and got all the CSS properties just right to play it. If there's a V2, I'll do more research and try out other animation methods. There are a lot of great tools out there for creating spritesheets because they're still a memory- and processing-light animation method.

The game has a new background but it's still stand-in art. I'm working on getting the plants into position and part of that is making them proportionate to the room they're in. This new background is pretty close to the proportions I have in mind.
